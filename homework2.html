Частина 3: Правила читання

    Поясни власними словами (українською) правила вимови наступних буквосполучень:
    ie: Довгий звук «і», як у слові "вода".
    chs: Звук, схожий на «кс», як у слові "оксид".
    s на початку слова: Зазвичай звучить як «з», наприклад, Sonne (сонце).
    Напиши по 5 прикладів для кожного правила читання (з перекладом).
    Приклад:
    Для ie:
    die Schule (школа)
    vier (чотири)
    Liebe (любов)
    …
    Для chs:
    wachsen (рости)
    Fuchs (лисиця)
    …
    Для s на початку:
    Sonne (сонце)
    Salz (сіль)
    …
